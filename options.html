Google Cloud Text-to-Speech (Premium Voices)

Enable high-quality Neural2 Hindi voices. Requires a Google Cloud API key with Text-to-Speech API enabled.

Steps: Go to Google Cloud Console → Create a project → Enable "Cloud Text-to-Speech API" → Create an API key under Credentials. Free tier: 250K chars/month (Neural2).

Google Cloud Translation API (Optional)

For higher quality translations, enter your Google Cloud Translation API key. Without it, the free Google Translate endpoint will be used.
